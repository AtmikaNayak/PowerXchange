# PowerXchange

PowerXchange is a student-focused book exchange platform that makes academic resources more affordable and accessible. The platform enables students to buy, sell, rent, and exchange books through a centralized marketplace, helping reduce costs and promote resource sharing within the student community.

## Live Demo

[PowerXchange](https://power-xchange.vercel.app)

## Features

* User authentication and account management
* Create and manage book listings
* Buy, sell, rent, and exchange books
* Search and browse available books
* Book request and exchange system
* User profile management
* Responsive user interface

## Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* Supabase

## Problem Statement

Students often spend a significant amount of money on academic books that are only used for a single semester. Finding reliable buyers, sellers, or exchange partners can also be difficult.

PowerXchange provides a dedicated platform where students can connect, trade books, and make educational resources more accessible and affordable.

## Project Highlights

* Simplifies the process of exchanging academic books
* Encourages reuse of educational resources
* Provides a centralized marketplace for students
* Reduces the cost of acquiring study materials
